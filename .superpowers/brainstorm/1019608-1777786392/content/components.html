Component Specifications

Eight core modules — each with a single responsibility and clear interface

1. SessionManager

Server Lifecycle + SDK Wrapper
// Exported API
interface SessionManager {
  // Lifecycle
  start(): Promise<void>          // spawns "opencode serve", waits for health
  stop(): Promise<void>           // kills server process, cleans up
  restart(): Promise<void>        // stop + start, preserves session list
  isRunning(): boolean

  // Session CRUD
  createSession(title?: string): Promise<Session>
  deleteSession(id: string): Promise<boolean>
  getSession(id: string): Promise<Session>
  listSessions(): Promise<Session[]>

  // Messaging
  sendPrompt(sessionId, parts: Part[], context?: ContextPackage): Promise<Message>
  sendCommand(sessionId, command: string): Promise<Message>
  abortSession(sessionId): Promise<boolean>

  // Events (SSE-backed)
  onEvent: vscode.EventEmitter<OpencodeEvent>  // tool_call, skill_load, thinking, etc.
  subscribe(): Promise<void>                     // connects to /event SSE stream
}
Port Management: Find a free port via getPort(), pass it to opencode serve --port {n}. Store port in workspace state. Auto-detect if an existing server is running on a known port.

2. ContextEngine

Intelligent Context Gathering (Worker Thread)
interface ContextPackage {
  openFiles:       { path, language, content (capped at 8k chars each) }[]
  activeSelection: { file, startLine, endLine, text } | null
  diagnostics:     { file, errors, warnings, hints }[]
  workspaceTree:   { name, type: "file"|"directory", depth: 3 }[]
  projectConfigs:  { type: "package.json"|"tsconfig"|..., content }[]
  gitStatus:       { branch, modified, staged, recentDiff? }
  terminalOutput:  { text } | null  // user-selected block
}

interface ContextEngine {
  gatherContext(config?: GatherConfig): Promise<ContextPackage>
  estimateTokens(pkg: ContextPackage): number  // tiktoken-based, <5ms
  onConfigChanged: vscode.Event<void>
}
Basic Mode
Open tabs + diagnostics + workspace tree. <10ms, always on.
Deep Mode
+ AST structure, git diff, project configs. <50ms, optional.

Rich Chat UI — The Primary Interaction Point

interface ChatMessage {
  role: "user" | "assistant" | "system"
  content: (TextBlock | ToolCallCard | SkillCard | DiffBlock | ContextBlock)[]
  timestamp: number
  sessionId: string
}
